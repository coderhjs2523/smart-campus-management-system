# 🎓 Smart Campus Management System

A modern web-based Smart Campus Management System designed to streamline campus operations, improve communication, and provide a centralized platform for students, faculty, and administrators.

## 📖 Overview

The Smart Campus Management System is a digital platform that automates various campus activities such as student management, attendance tracking, notices, events, and academic records. The goal is to create a smart and efficient campus environment by reducing manual work and improving accessibility.

## ✨ Features

### 👨‍🎓 Student Module
- Student Registration & Login
- Profile Management
- View Attendance Records
- Access Notices & Announcements
- View Academic Information

### 👨‍🏫 Faculty Module

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/smart-campus.git
```

### Navigate to Project Folder

```bash
cd smart-campus
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```

### Run Development Server

```bash
npm run dev
```
